## Steps to run this system after cloning

-   run `composer install`
-   run `cp .env.example. .env`
-   run `php artisan key:generate`
-   run `php artisan migrate`
-   run `php artisan db:seed --class=AdminTableSeeder`
-   run `npm install`
-   run `npm run dev`
-   php `artisan optimize`
-   php `artisan serve`
